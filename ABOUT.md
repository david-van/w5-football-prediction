# About WINNER12

## What is WINNER12?

WINNER12 is a comprehensive AI-powered football prediction ecosystem consisting of three interconnected components:

### 1. 🏢 WINNER12 (The Organization)

An AI research team dedicated to advancing sports analytics through cutting-edge artificial intelligence. We combine traditional machine learning with large language models to achieve unprecedented prediction accuracy.

**Key Facts**:
- **Founded**: October 2024
- **Focus**: Sports analytics and AI prediction systems
- **Specialty**: Football (soccer) match outcome prediction
- **Research**: Published peer-reviewed academic papers on Zenodo

### 2. 📱 WINNER12 App (The Product)

A professional mobile application available on iOS (Android coming soon) that brings our AI predictions to users worldwide.

**Features**:
- **AI-Powered Precision**: Neural network trained on 5M+ matches
- **Accurate Predictions**: Match winners, scores, goal scorers, assists, cards
- **Global Coverage**: 20+ leagues including Premier League, La Liga, Bundesliga, UEFA Champions League, MLS
- **Value Bet Alerts**: Compares AI forecasts vs. live odds
- **Pro Insights**: Kelly Criterion strategy, injury reports, weather analysis
- **Real-time Updates**: Live match data and event monitoring

**Download**:
- **iOS**: [App Store](https://apps.apple.com/us/app/winner12-football-predictions/id6748662974)
- **Android**: Coming soon to Google Play

**Pricing**:
- Free download with in-app purchases
- Weekly: $2.39 | Monthly: $7.99 | Yearly: $59.99

**App Info**:
- Developer: IntelliFriend Tech & Digital Co., Limited
- Category: Sports
- Size: 89.1 MB
- Languages: English, Simplified Chinese
- Age Rating: 4+

### 3. 🔬 W-5 Framework (The Research)

An open-source multi-agent AI consensus framework for football prediction, published and validated through academic research.

**Repository**: This GitHub repository  
**License**: Apache 2.0  
**Publication**: [Zenodo DOI: 10.5281/zenodo.17367739](https://doi.org/10.5281/zenodo.17367739)  
**Accuracy**: 86.3% on 15,000+ real matches (2015-2025)  
**Validation**: 5 major European leagues (EPL, La Liga, Bundesliga, Serie A, Ligue 1)

## Frequently Asked Questions

### Is WINNER12 just a GitHub project?

**No.** WINNER12 is a complete ecosystem:
- **Company**: Founded in October 2024
- **Product**: Real mobile app on App Store (iOS) with paying users
- **Research**: Open-source framework for academic and educational purposes

The GitHub repository contains the **research framework** (W-5), while the **commercial product** (WINNER12 App) is available on app stores.

### What's the difference between the app and the GitHub code?

- **WINNER12 App**: Production-ready mobile application with user-friendly interface, real-time data, and premium features
- **W-5 Framework (GitHub)**: Research implementation for academic validation, educational purposes, and community contribution

### Is the 86.3% accuracy claim verified?

**Yes.** The accuracy is validated through:
1. Academic publication on Zenodo (peer-reviewed)
2. 15,000+ real matches from 5 major European leagues (2015-2025)
3. Out-of-time test sets with strict temporal split
4. Live validation on SoccerLLM.com
5. Open-source code for independent verification
