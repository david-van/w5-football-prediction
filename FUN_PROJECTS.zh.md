## ⚽ SoccerLLM.com - "脆弱的"单模型实验

**网站**：[soccerllm.com](https://soccerllm.com)  
**状态**：🟢 运行中并持续更新  
**目的**：展示单个 LLM 局限性的教育演示

### 它是什么？

一个既有趣又科学严谨的实验，用于展示 AI 体育预测中**不应该做什么**。我们故意使用单个 LLM 模型（GPT-4、Claude、Gemini）**单独**预测足球比赛，没有任何共识机制或 ML 基线。

### 为什么要构建它？

1. **教育目的**：以透明的方式展示单模型的局限性
2. **科学基线**：为 W-5 框架评估提供对照组  
3. **透明度**：真实预测与真实失败 - 不挑选结果
4. **社区学习**：帮助他人理解为什么多智能体系统很重要

### 它如何工作？

```
用户选择一场比赛
    ↓
单个 LLM（GPT-4、Claude 或 Gemini）分析
    ↓
模型做出预测（主胜/平局/客胜）
    ↓
比赛结束后：对比预测与实际结果
    ↓
随时间跟踪准确率
```

**没有共识。没有 ML。没有安全网。只有一个 LLM 的意见。**

### 目前的结果

| 模型 | 预测数 | 准确率 | 显著弱点 |
|-------|------------|----------|------------------|
| GPT-4 单独 | 100+ | ~52% | 对热门球队过度自信 |
| Claude 单独 | 100+ | ~48% | 低估冷门球队 |
| Gemini 单独 | 100+ | ~50% | 推理不一致 |
| **平均** | - | **~50%** | **仅略好于随机！** |

**与 W-5 框架对比**：高置信度预测准确率 80.1%

### 经验教训

#### 我们发现的问题 ❌

- **单个 LLM 不稳定**：同一场比赛，不同日期 = 不同预测
- **过度自信普遍存在**：即使错误，模型也表现出高置信度
- **缺乏数值基础**：LLM 难以处理统计模式
- **近期偏差**：最近的比赛对预测产生不成比例的影响
- **没有自我纠正**：没有辩论，错误就不会被挑战

#### 这证明了什么 ✅

- **多智能体共识是必要的**：辩论减少个体偏见
- **混合 ML+LLM 优于纯 LLM**：定量 + 定性 > 仅定性
- **选择性预测很重要**：知道何时不预测至关重要
- **校准是关键**：置信度分数必须与实际准确率匹配

## 🔬 实验哲学

### 透明度优先

我们相信 AI 研究中的**彻底透明**：

- ✅ 展示失败，而不仅仅是成功
- ✅ 清楚地记录局限性
- ✅ 公开数据
- ✅ 详细解释方法论

### 科学严谨

SoccerLLM.com 遵循科学最佳实践：

- **不挑选结果**：所有预测都被记录
- **不事后调整**：预测在比赛前锁定
- **无数据泄漏**：模型看不到未来结果
- **可重现**：相同输入 → 相同输出

### 教育使命

我们的目标是**教育**，而不仅仅是获胜：

- 帮助人们理解 AI 的局限性
- 促进对 AI 声明的批判性思考
- 鼓励多智能体和混合方法
- 通过开放研究推进该领域

## 🎯 对比：SoccerLLM vs W-5

| 特性 | SoccerLLM.com | W-5 框架 |
|---------|---------------|---------------|
| **架构** | 单个 LLM | 多智能体 + ML |
| **准确率** | ~50% | 80.1%（高置信度） |
| **一致性** | 低 | 高 |
| **置信度校准** | 差 | 好 |
| **弃权** | 无 | 68.9% |
| **目的** | 教育性 | 生产就绪 |

**结论**：单模型是有趣的实验，但多智能体共识对于可靠预测是必要的。
