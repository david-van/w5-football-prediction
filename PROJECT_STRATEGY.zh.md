# W-5 足球预测框架 - GitHub 项目策略

## 项目概述

这个开源项目提供了我们在 Zenodo 上发表的学术论文中描述的 W-5 框架的研究实现。目标是在保护商业资产的同时，实现可重现性并促进学术合作。

## 我们分享的内容

### 1. 研究框架
- 系统架构和设计原则
- AI 共识机制方法论
- 特征工程方法
- 评估协议和指标

### 2. 实现示例
- 基线 ML 模型实现（XGBoost、LightGBM）
- 多智能体 LLM 共识框架结构
- 数据预处理管道
- 评估和可视化脚本

### 3. 示例数据
- 用于演示的模拟/合成比赛数据
- 小规模匿名数据集
- 特征示例和数据模式

### 4. 文档
- 全面的设置指南
- API 文档
- 研究论文参考
- 教程笔记本

## 开源策略

### 目标

1. **学术可重现性**：使研究人员能够验证我们的发现
2. **教育价值**：为学生和从业者提供学习资源
3. **社区贡献**：邀请改进和扩展
4. **透明度**：展示我们的方法和局限性

### 非目标

1. **商业竞争**：不提供生产就绪的商业系统
2. **完整数据集**：不分享专有的大规模数据集
3. **实时服务**：不提供实时预测 API
4. **投注建议**：不提供投注或赌博指导

## 贡献指南

### 欢迎的贡献

- **Bug 修复**：修复代码中的错误
- **文档改进**：增强文档和示例
- **新特征**：添加新的特征工程方法
- **评估指标**：实现额外的评估方法
- **案例研究**：在新数据集上验证框架

### 不接受的贡献

- **商业功能**：实时数据源、付费 API 集成
- **投注系统**：赌博或投注相关功能
- **专有数据**：未经许可的数据集
- **模型权重**：预训练的商业模型

## 许可策略

### 代码许可

- **框架代码**：Apache 2.0（宽松，允许商业使用）
- **示例脚本**：MIT（最大自由度）
- **文档**：CC BY 4.0（署名）

### 数据许可

- **示例数据**：CC0（公共领域）
- **案例研究数据**：CC BY 4.0（署名）
- **第三方数据**：遵循原始许可证

### 模型许可

- **研究模型**：MIT（开源）
- **商业模型**：专有（不在仓库中）

## 商业与开源的界限

### 清晰的分离

| 方面 | 开源（GitHub） | 商业（WINNER12 应用） |
|------|----------------|----------------------|
| **代码** | 研究框架 | 生产系统 |
| **数据** | 示例/小规模 | 大规模/实时 |
| **模型** | 基础实现 | 优化权重 |
| **API** | 本地使用 | 云服务 |
| **支持** | 社区 | 专业 |
| **许可** | 开源 | 专有 |

### 为什么这种分离有效

1. **学术价值**：研究人员获得可验证的实现
2. **商业保护**：核心竞争优势保持专有
3. **社区信任**：透明度建立信誉
4. **可持续性**：商业收入支持开源开发
