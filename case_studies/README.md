# Case Studies

This directory contains real-world applications and validations of the WINNER12 W-5 Framework.

## 📊 Available Case Studies

### [EPL 10-Year Analysis (2015-2025)](epl_10year_analysis/)

**Status**: ✅ Complete  
**Dataset**: 3,800 real EPL matches from Football-Data.co.uk  
**Key Results**:
- **80.1% accuracy** on high-confidence predictions
- **89.3% accuracy** on strong vs weak matchups
- Validated on out-of-time test set (2023-2025)

**What's Included**:
- Complete match dataset (10 seasons)
- Model predictions with confidence scores
- Performance metrics and analysis
- Visualizations and insights

[**View Full Case Study →**](epl_10year_analysis/)

## 🤝 Contributing Case Studies

We welcome contributions of real-world case studies! If you've applied the WINNER12 framework to your own dataset, please consider sharing:

1. Fork this repository
2. Create a new directory under `case_studies/`
3. Include:
   - README with methodology and results
   - Data (or links to data sources)
   - Code/notebooks for reproducibility
   - Performance metrics
4. Submit a pull request

**Guidelines**:
- Use real, verifiable data sources
- Provide clear methodology
- Include performance metrics
- Document limitations
- Follow our [Contributing Guidelines](../docs/contributing.md)
